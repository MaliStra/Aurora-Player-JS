Aurora Player — это мощный веб-аудиоплеер с расширенными возможностями обработки звука: эквалайзер, эффекты, кроссовер, звуковое поле и многое другое. Плеер работает полностью в браузере, не требует установки и поддерживает большинство аудиоформатов (MP3, M4A, OGG, WAV и др.).

Этот плеер создан для настройки автомагнитол, особенно на Android, где часто отсутствуют удобные настройки для музыки в автомобиле. Большая часть функций сосредоточена на настройке звука и работе с сабвуфером. Проект еще не доработан, но уже сейчас доступен для просмотра его возможностей. Он написан на чистом JavaScript.

🚀 Установка и запуск
Скачайте или клонируйте репозиторий:

Запустите локальный веб-сервер (например, через Live Server в VS Code или с помощью python -m http.server).

Откройте в браузере http://localhost:8000 (или другой адрес, в зависимости от сервера).

Примечание: для извлечения обложек из аудиофайлов используется библиотека jsmediatags. Она подключается по CDN, поэтому интернет-соединение необходимо.

🎮 Использование
Добавление треков
Нажмите «Добавить» → выберите один или несколько аудиофайлов.

Нажмите «Папку» → выберите папку (будут добавлены все аудиофайлы во всех подпапках).

Перетащите файлы или папку в окно плеера.

Управление воспроизведением
▶️ Play / Pause – кнопка в центре или пробел.

⏮️ Предыдущий / Следующий – кнопки или стрелки влево/вправо (Shift+стрелка для переключения трека).

🔀 Перемешать – случайный порядок.

🔁 Повтор – один клик – повтор списка, два клика – повтор одного трека.

Эквалайзер и эффекты
Нажмите FX, чтобы открыть панель эффектов. Доступны вкладки:

Эквалайзер – регулировка 10 полос, выбор пресета.

Эффекты – скорость, баланс, бас-буст, эхо.

Продвинутые – компрессор, реверберация, дисторшн, фильтры, тремоло, караоке, автоусиление.

Кроссовер – 2-полосный кроссовер, полосовой фильтр, time alignment, DSP.

Звук. поле – регулировка каналов, пресеты для автомобиля, сабвуфер.

Громкость
Ползунок громкости или клавиши ↑ / ↓.

M – отключить звук.

Плейлист
Клик по треку – воспроизведение.

Удалить трек можно по кнопке × справа.

Если у трека есть обложка, она отображается в центре визуализатора.
